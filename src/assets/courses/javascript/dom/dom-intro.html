    How to Find HTML Elements

    Following selectors will help to find the HTML elemnts.

  • using ID
  • using classname
  • using Tagname
  • Queryselector
    • Property Description
    element = document.getElementById('id_name'); using ID
    element = document.getElementsByClassName('class_name'); Using Class name.It will return all elements which have same class name in the document/HTML page.
    element = document.getElementsByTagName('tag_name'); Using Tagname
    element = document.querySelector('selector_name');
    element = document.querySelectorAll('selector_name');    		 Using QuerySelector

    How to change a HTML Elements

    After finding elements you can change the content, attributes, Css styles and can set Attribute to the perticular element.

    Property Description
    element.innerHTML = new HTML content; Change innerHTML
    element.attribute = new value; Change Attribute value
    element.style.property = new style; Change CSS Style
    element.setAttribute(attribute,value); Set New Attribute to an Element

    Add/Delete HTML Element

    Property Description
    document.createElement(element); create an HTML Element
    document.removeChild(element); remove an HTML element
    document.appendElement(element); add an HTML Element
    document.replaceChild(new, old); Replace an HTML Element

    Event Handler

    Frequently user events

    Event names Description
    Onload trigger an event when the particulat element is loaded in HTML page.
    Onclick trigger an event by clicking an element in HTML page.
    Onkeyup trigger an event by when a key pressed.event triggered when the key is released.
    Onmouseover Trigger an event when mouse enter to the particular element in HTML page
    Onfocus trigger an event when particular element get focused in HTML page.
