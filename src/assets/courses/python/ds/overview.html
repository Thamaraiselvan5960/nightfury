DataStructure overview

Python is a high level object oriented programming language. Also it is an interpreted and dynamically typed language. So we need not to mention the data structure while we create any variable in Python. Python automatically assigns the data structure, but how will Python know which data structure is going to create?

Here we will see what are the data structures available in Python and how it works.

DataTypes in Python is broadly classified into 2 types.

Primitive and Non primitive

  • Primitive (string, numerical(int, float), boolean)
  • Non Primitive (List, Tuple, Set, Dictionary)

    • String

    String is a combination of alphabetical characters, numbers and symbols. It should be enclosed by quotation.

    Example:

  • ‘I am Batman’
  • “I’m Batman”

    • Numerical Types

    Integer

    If we are assigning a rounded number like 5 or 8, Python will take this as an integer.

    Example

  • num = 5
  • num_2 = 56

    • Float

    As like integer, float is also a numerical data type, by this will have a decimal point in it. You can even create a whole number like5 as float by add a point (eg: 5.0)

    Example

    float_var = 5.6

    float_var2 = 8.0

    Groups/ Arrays

    List

    List is used for storing the objects together, kind of an array. In a list, we can store any type of data structure.

    Tuple

    Tuple is similar to list. But once we create a tuple, we can’t able to change anything after that.

    Set

    Set is an unordered array. Also in set, there won’t be any duplicate values.

    Dictionary

    Dictionary is made up of keys and values. This key value pair together is considered as a single element.

    Boolean

    Boolean is having only two values. True and False. This is mainly used for condition checking.
